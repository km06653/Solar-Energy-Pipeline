# Solar Energy Data Pipeline Architecture


## Overview

This project builds an end-to-end ETL pipeline for solar energy analytics.

Raw solar generation data is extracted from CSV files, cleaned and validated using Python, loaded into a PostgreSQL database, and visualized through Power BI dashboards.
